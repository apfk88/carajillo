# Carajillo

A beautiful static site celebrating the carajillo cocktail.

## Deploy to Vercel

1. Push this repo to GitHub
2. Import the repo in [Vercel](https://vercel.com/new)
3. Enable Analytics in the project
4. Deploy

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Vercel Analytics

This site includes the static HTML Vercel Analytics snippet. Once Analytics is enabled in Vercel and the site is deployed there, page views should start flowing automatically.

## Local Development

Just open `index.html` in your browser, or use a local server:

```bash
npx serve .
```

## About

The carajillo is a Spanish coffee cocktail made with espresso and Licor 43. Simple, delicious, timeless.
